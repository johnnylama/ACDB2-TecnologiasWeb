¿Qué es el desarrollo web?

El desarrollo web es el proceso de crear sitios y aplicaciones que se ejecutan en un navegador. Combina estructura, estilo y comportamiento para entregar experiencias claras, accesibles y funcionales.

Las buenas practicas incluyen contenido semantico, diseño responsivo, rendimiento optimizado y accesibilidad basica.
