¿Qué es el desarrollo web?

El desarrollo web es el proceso de crear sitios y aplicaciones que funcionan dentro de un navegador. Para lograrlo, se utilizan distintos lenguajes que cumplen roles específicos: HTML define la estructura, CSS aplica el diseño visual y JavaScript añade interactividad.

Estos lenguajes son la base de prácticamente todos los sitios que visitamos en la World Wide Web, desde blogs sencillos hasta plataformas de comercio electrónico y redes sociales.
