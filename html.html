¿Qué es HTML?

HTML es el lenguaje estandar para crear la estructura de las paginas web. Usa etiquetas para indicar al navegador si el contenido es un titulo, un parrafo, una imagen, un enlace o un formulario.

HTML5 incorpora etiquetas semanticas que mejoran la organizacion del contenido y la accesibilidad.

Consejo: usa etiquetas como <header>, <main> y <footer> para que el contenido sea mas facil de comprender.

Etiquetas clave

  • <header> Encabezado principal.
  • <nav> Menu de navegacion.
  • <section> Secciones tematicas.
  • <article> Contenido independiente.
  • <footer> Pie de pagina.

Ejemplo basico de HTML5

Una estructura minima con encabezado y contenido principal.

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi primera pagina</title>
  </head>
  <body>
    <header>
      <h1>Hola mundo</h1>
    </header>
    <main>Contenido principal</main>
  </body>
</html>

Caracteristicas principales

  • Define la estructura del documento.
  • Permite combinar texto, imagenes, audio y video.
  • Trabaja junto con CSS y JavaScript.
  • Es un estandar mantenido por el W3C.
