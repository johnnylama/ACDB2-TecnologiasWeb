¿Qué es HTML?

HTML es el lenguaje estándar para crear la estructura de las páginas web. Utiliza etiquetas para indicar al navegador qué tipo de contenido se está mostrando, por ejemplo: títulos, párrafos, imágenes, enlaces, tablas, formularios, entre otros.

La versión actual recomendada es HTML5, que incorpora nuevas etiquetas semánticas como <header>, <nav>, <section>, <article> y <footer>, las cuales mejoran la organización del contenido y la accesibilidad.

Ejemplo sencillo de código HTML5

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>Hola mundo</h1>
  </body>
</html>

Características principales de HTML

  1. Define la estructura del documento.
  2. Permite combinar texto, imágenes, audio y video.
  3. Trabaja junto con CSS y JavaScript.
  4. Es independiente del sistema operativo y del navegador.
  5. Es un estándar mantenido por el W3C.
