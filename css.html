CSS

Lenguaje de estilos que transforma estructuras en experiencias visuales.

Siguiente: JavaScript

¿Qué es CSS?

CSS define colores, tipografias, espaciados y layouts para que una pagina HTML se vea coherente en distintos dispositivos.

Con CSS puedes crear grillas flexibles, animaciones y adaptaciones responsivas sin modificar el contenido base.

Separar estructura y presentacion facilita el mantenimiento y la escalabilidad.

Ventajas

  • Reutilizacion de estilos en varias paginas.
  • Diseno adaptable a movil y escritorio.
  • Consistencia visual en todo el sitio.
  • Mejor experiencia de usuario.

Ejemplo basico de reglas CSS

Una regla CSS combina selectores y declaraciones.

h1 {
  font-family: "Fraunces", serif;
  text-align: center;
  color: #1d6f8a;
}

p {
  line-height: 1.6;
  color: #556572;
}

Conceptos basicos

Indican a que elementos HTML se aplicara un estilo.

Cada elemento tiene contenido, padding, borde y margen.

Sistemas modernos para organizar columnas y filas responsivas.

Buenas practicas

Jerarquia clara

Define tipografias y espaciados consistentes.

Mobile first

Diseña pensando primero en pantallas pequenas.

Colores consistentes

Usa variables CSS para mantener una paleta estable.

Animaciones utiles

Aplica transiciones suaves solo cuando aporten valor.
