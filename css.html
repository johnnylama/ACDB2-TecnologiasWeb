¿Qué es CSS?

CSS es el lenguaje que se utiliza para definir la apariencia visual de un sitio web. Permite controlar colores, tamaños, márgenes, tipografía y la disposición de los elementos en la pantalla.

Gracias a CSS, un mismo documento HTML puede mostrarse de formas muy diferentes, adaptándose a móviles, tabletas y computadores de escritorio.

Ejemplo básico de reglas CSS

h1 {
  font-family: Arial, sans-serif;
  text-align: center;
}

p {
  line-height: 1.5;
}

Ventajas de utilizar CSS

  • Separa el contenido (HTML) de la presentación.
  • Permite reutilizar estilos en múltiples páginas.
  • Reduce la cantidad de código repetido.
  • Facilita el diseño adaptable a diferentes dispositivos.
  • Mejora la experiencia de usuario y la accesibilidad.

Conceptos básicos de CSS

Una regla CSS está compuesta por un selector y un conjunto de declaraciones:

  1. El selector indica a qué elementos HTML se aplicará el estilo.
  2. Cada declaración contiene una propiedad y un valor.
